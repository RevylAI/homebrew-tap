# Homebrew Tap for Revyl

This is the official Homebrew tap for [Revyl CLI](https://github.com/RevylAI/revyl-cli).

## Installation

```bash
brew install RevylAI/tap/revyl
```

## Usage

After installation, you can use the `revyl` command:

```bash
revyl --help
revyl version
```

## Updating

```bash
brew upgrade revyl
```

## Uninstalling

```bash
brew uninstall revyl
```
